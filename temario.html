Estructura del Proyecto y Sustentación

Este proyecto busca enseñar Blender de forma didáctica con una estructura de unidades bien definida.

La línea de tiempo interactiva permite navegar por los temas de forma intuitiva.

Sustentación de la estructura del HTML y CSS

1. Estructura general del HTML

Encabezado (<head>)

  • Define la codificación (UTF-8) y asegura la compatibilidad con dispositivos (viewport).
  • Importa la fuente Poppins desde Google Fonts.
  • Incluye estilos CSS embebidos en la misma página.

Cuerpo (<body>)

  • Se usa flexbox en el body para centrar elementos y mantener una estructura fluida.
  • Contiene una cabecera de línea de tiempo (.timeline-header) con animaciones en el borde.
  • Un div.timeline-container que contiene una línea de tiempo (.timeline) y varios elementos .item alineados a la izquierda y derecha.
  • Un footer con créditos y metadatos sobre el proyecto.
  • Un popup modal con video para futuras interacciones.

2. Justificación de la estructura CSS

Fuentes y colores:

  • Se elige Poppins por su legibilidad, y se usa una paleta de grises y azules para dar un diseño profesional.

Efectos visuales y animaciones:

  • Se aplica @keyframes animate-border en .timeline-header y .footer, dándoles un efecto de borde dinámico.
  • Los .item tienen transiciones suaves y un hover que resalta la interactividad.
  • .popup-container tiene un fadeIn para una mejor experiencia visual.

Estructura de la línea de tiempo (.timeline)

  • position: absolute mantiene la línea central fija en el medio.
  • .dot representa los puntos de eventos en la línea de tiempo.
  • .item usa flexbox y position: relative para alinear los contenidos de forma correcta.
